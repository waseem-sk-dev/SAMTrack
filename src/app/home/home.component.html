About SAMTrack

SAMTrack is a comprehensive web application for managing attendance and student records. Features include:

  • User and role management
  • Student and subject tracking
  • Attendance by faculty
  • Admin/faculty dashboards

Built using Angular and Spring Boot for real-world deployment.
